Requirements:

   * Java 8
   * ChromeDriver    
    
    
        Mac users with Homebrew installed:  brew cask install chromedriver
        Debian based Linux distros: sudo apt-get install chromium-chromedriver
        Windows users with Chocolatey installed: choco install chromedriver
